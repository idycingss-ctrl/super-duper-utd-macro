# super-duper-utd-macro
A macro for the roblox game [universal tower defense](https://www.roblox.com/games/133410800847665/Universal-Tower-Defense) primarily made using Autohotkey

## Features
- custom strategies
- auto rejoin
- webhook logging
- state detection

## Requirements
- Windows 10 or older
- Autohotkey v1.1
- 1920x1080 screen
- Fullscreened

## Instructions
1. Downloading
    - Download Autohotkey v1.1
    - Clone this repo or download the source code from releases tab
    - Extract the folder
2. Using the macro
    - Open GameMacro.ahk
    - Go into any map in the game
    - Press F3 to open the gui
    - In the keys tab, bind your units to the keys
    - In the coordinates tab, press f6 to angle camera and then select the coordinates
    - In the strategies tab, create the strategy
    - Save and close the window
    - Press F5 to run

## Functions
- F3: Show GUI
- F5: Run Macro
- F6: Angle Camera
- F7: Webhook Debug
- F10: Pause Macro
- F11: Close Macro
